# Инструкция по использованию калькулятора

## Введение
Данный калькулятор позволяет выполнять базовые арифметические операции: сложение, вычитание, умножение и деление.

## Установка
1. Склонируйте репозиторий:
   ```
git clone https://github.com/bbrusentseva-b/test-d.git
   ```
2. Откройте файл `index.html` в веб-браузере.

## Использование
1. Введите первое число в поле ввода.
2. Выберите операцию: сложение (+), вычитание (-), умножение (×), деление (÷).
3. Введите второе число в поле ввода.
4. Нажмите кнопку "Равно (=)", чтобы получить результат.
5. Результат будет отображён на экране.

## Примечания
- Убедитесь, что вы вводите числа правильно, особенно при делении (деление на ноль не допускается).
- Освежите страницу, если необходимо начать с нуля.

## Заключение
Калькулятор является простым инструментом для выполнения базовых вычислений.
